Vela provides the shared documentation shell: navigation, typography, theme and language controls, code block styling, sidebar polish, and Astro defaults.

## What stays in the theme

- Documentation layout and navigation polish.
- Shared typography, colors, code blocks, tables, cards, and sidebars.
- Theme and language dropdown components.
- Optional utility components such as `Terminal` and `LanguageRedirect`.

## What stays in each project

- Product-specific content and information architecture.
- Homepage overrides when a project needs a fully custom landing page.
- Project-specific examples, CTA blocks, and marketing sections.
